00044 int usbOpenDevice(usb_dev_handle **device, int vendorID, char *vendorNamePattern, int productID, char *productNamePattern, char *serialNamePattern, FILE *printMatchingDevicesFp, FILE *warningsFp);
00045 /* This function iterates over all devices on all USB busses and searches for
00046  * a device. Matching is done first by means of Vendor- and Product-ID (passed
00047  * in 'vendorID' and 'productID'. An ID of 0 matches any numeric ID (wildcard).
00048  * When a device matches by its IDs, matching by names is performed. Name
00049  * matching can be done on textual vendor name ('vendorNamePattern'), product
00050  * name ('productNamePattern') and serial number ('serialNamePattern'). A
00051  * device matches only if all non-null pattern match. If you don't care about
00052  * a string, pass NULL for the pattern. Patterns are Unix shell style pattern:
00053  * '*' stands for 0 or more characters, '?' for one single character, a list
00054  * of characters in square brackets for a single character from the list
00055  * (dashes are allowed to specify a range) and if the lis of characters begins
00056  * with a caret ('^'), it matches one character which is NOT in the list.
00057  * Other parameters to the function: If 'warningsFp' is not NULL, warning
00058  * messages are printed to this file descriptor with fprintf(). If
00059  * 'printMatchingDevicesFp' is not NULL, no device is opened but matching
00060  * devices are printed to the given file descriptor with fprintf().
00061  * If a device is opened, the resulting USB handle is stored in '*device'. A
00062  * pointer to a "usb_dev_handle *" type variable must be passed here.
00063  * Returns: 0 on success, an error code (see defines below) on failure.
00064  */
00065
